Allgemeine Hinweise

Diese Datenschutzerklärung informiert Sie über die Art, den Umfang und den Zweck der Verarbeitung von personenbezogenen Daten auf dieser Webseite.

Hosting durch GitHub Pages

Diese Webseite wird als GitHub Page gehostet. Anbieter ist die GitHub Inc., 88 Colin P. Kelly Jr. St, San Francisco, CA 94107, USA.

Beim Besuch dieser Webseite werden durch GitHub automatisch Informationen erhoben und in so genannten Server-Log-Files gespeichert. Dazu gehören:

  • IP-Adresse des zugreifenden Rechners
  • Datum und Uhrzeit des Abrufs
  • Name und URL der abgerufenen Datei
  • Referrer-URL (die zuvor besuchte Seite)
  • Verwendeter Browser und ggf. das Betriebssystem Ihres Rechners

Die Datenverarbeitung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO zur Gewährleistung eines reibungslosen Verbindungsaufbaus und der Systemsicherheit. GitHub verarbeitet diese Daten in den USA; Grundlage hierfür sind die Standardvertragsklauseln der EU-Kommission.

Speicherung von Spracheinstellungen (localStorage)

Diese Webseite nutzt den sogenannten localStorage, um Ihre Sprachpräferenz (Deutsch oder Englisch) lokal auf Ihrem Endgerät zu speichern.

  • Zweck: Bereitstellung der Webseite in der vom Nutzer gewählten Sprache bei erneutem Aufruf.
  • Rechtsgrundlage: Art. 6 Abs. 1 lit. f DSGVO (Berechtigtes Interesse an der Benutzerfreundlichkeit).
  • Speicherdauer: Die Daten verbleiben in Ihrem Browser, bis Sie den Cache Ihres Browsers leeren oder die lokale Speicherung manuell löschen.
